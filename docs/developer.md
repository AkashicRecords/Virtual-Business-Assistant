# Developer Documentation

## Setup Development Environment
1. Clone repository
2. Install dependencies
3. Configure services
4. Run tests

## Project Structure
```
gmail-assistant/
├── services/
│   ├── email_service.py
│   ├── llm_service.py
│   └── voice_service.py
├── gui/
│   └── main_window.py
├── tests/
│   └── test_*.py
└── config/
    └── settings.py
```

## Development Guidelines
- Code Style: PEP 8
- Documentation: Google style docstrings
- Testing: pytest
- Logging: structured logging
